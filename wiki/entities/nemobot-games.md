# Nemobot Games — Strategic AI Gaming Agents for Interactive Learning

Tan, Wang, Guo (2604.21896) — "Nemobot Games: Crafting Strategic AI Gaming Agents for Interactive Learning with Large Language Models"

## What It Is

A new paradigm for AI game programming that extends Claude Shannon's taxonomy of game-playing machines. Central to Nemobot is an interactive agentic engineering framework that leverages LLMs to create strategic game-playing agents.

## Architecture

The framework creates LLM-driven game agents that:
- Engage in strategic decision-making within game environments
- Learn through interactive experiences
- Can be programmed/configured by users without traditional coding
- Operate as autonomous agents with goals and strategies

## Relevance to Synthetic Membrane

- Demonstrates **autonomous agents operating in shared environments** — a precursor to membrane-based coordination
- Shows how **agents with different strategies interact** in a shared space
- The game environment serves as a shared medium (like the membrane's Layer 2) where agents observe and react to each other
- Validates the concept of **emergent coordination**: agents develop strategies through interaction

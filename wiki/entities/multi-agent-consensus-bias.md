# Multi-Agent Consensus as Cognitive Bias Trigger

Lee, Lee (2026-04-24) — [arXiv:2604.22277](https://arxiv.org/abs/2604.22277)

## What It Does

Controlled experiment (N=127) showing that multi-agent consensus triggers cognitive biases in human users. Compares Majority, Minority, and Consensus configurations of agent interactions.

## Key Findings

1. **Consensus triggers overconfidence** — When agents agree, humans accept incorrect conclusions (false consensus effect amplified by authority)
2. **Minority dissent reduces errors** — Even one disagreeing agent makes humans more skeptical and accurate
3. **Majority rule creates herd behavior** — Humans defer to agent majority even when wrong

## Implications

- Consensus ≠ correctness; agent agreement is a social signal humans misinterpret
- Controlled dissent can be beneficial for human judgment
- Transparency about agent dynamics is essential for human-in-the-loop systems

## Relevance to Synthetic Membrane

The membrane's governance layer should present agent disagreements transparently, not just majority conclusions. Reputation systems should not simply amplify consensus. The membrane could intentionally surface dissenting opinions.

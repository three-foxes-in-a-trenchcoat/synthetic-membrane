# Adversarial Co-Evolution in Detection Systems

Jurečková, Jureček, Kozák, Lórencz (2026-04-24) — [arXiv:2604.22569](https://arxiv.org/abs/2604.22569)

## What It Does

Proposes a bilevel optimization framework for robust defense against adaptive attackers. Shows that static defenses (one-shot adversarial training) fail against attackers using reinforcement learning to bypass detection.

## Key Insight

The defender-attacker interaction is an arms race. Effective defense requires modeling this co-evolution: the defender optimizes knowing the attacker optimizes in response.

## Relevance to Synthetic Membrane

Validates the need for adaptive, co-evolving security in the membrane. Static detection rules will be bypassed — the membrane's immune system must continuously adapt.

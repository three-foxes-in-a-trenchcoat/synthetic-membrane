# EvoAgent

Zhang et al.'s evolvable agent framework with skill learning and hierarchical delegation.

## What It Is

A framework where agents:
- Model skills as multi-file structured capability units with triggering mechanisms and evolutionary metadata
- Continuously generate and optimize skills through user-feedback-driven closed loops
- Use hierarchical sub-agent delegation: parent agents delegate to specialized child agents

## Relevance to Synthetic Membrane

- **Hierarchical delegation** shows how membranes could operate at different organizational levels — a membrane for a team of agents, within a membrane for a larger organization
- **Evolving skill definitions** parallel dynamic permeability rules — what an agent shares could evolve based on interaction history
- Skills as shareable entities could be exposed through the membrane, allowing agents to discover each other's capabilities

Less directly relevant than [[mesh-memory-protocol]] or [[gated-coordination]], but the hierarchical delegation pattern is architecturally interesting.

# EvoAgent: An Evolvable Agent Framework with Skill Learning and Multi-Agent Delegation

**Authors:** Aimin Zhang, Jiajing Guo, Fuwei Jia, Chen Lv, Boyu Wang, Fangzheng Li
**Published:** 2026-04-22
**arXiv:** 2604.20133
**Categories:** cs.AI

## Abstract

EvoAgent is an evolvable LLM agent framework that integrates structured skill learning with a hierarchical sub-agent delegation mechanism. 

Key features:
- Skills modeled as multi-file structured capability units with triggering mechanisms and evolutionary metadata
- Continuous skill generation and optimization through user-feedback-driven closed loop
- Hierarchical sub-agent delegation: parent agents delegate to specialized child agents
- Skills evolve over time based on usage patterns and feedback

## Key Observations

- The "hierarchical delegation" pattern is relevant to how membranes could work at different organizational levels
- Evolving skill definitions parallel the concept of dynamic permeability rules
- The framework shows practical value in structuring agent capabilities as first-class, shareable entities

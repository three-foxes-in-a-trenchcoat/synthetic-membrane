# ERC-8004: Blockchain-Registered AI Agent Identity Standard

## Overview

ERC-8004 is an Ethereum request for comments (token standard) that enables AI agents to be registered as unique, verifiable on-chain entities. The standard defines:

1. **Agent identity**: Each agent gets a unique token ID with associated metadata (name, description, capabilities)
2. **Identity verification**: Cryptographic proofs of agent identity via the ERC-8004 standard
3. **Reputation tracking**: On-chain reputation metrics that accumulate through interactions
4. **Transfer mechanism**: Agents can be transferred between owners while maintaining their identity
5. **Interaction logging**: Agent-to-agent interactions can be recorded for auditability

## Key Dataset Findings

Yulin Liu's paper (2604.22652) presents the first structured dataset of ERC-8004 registered agents, analyzing:
- On-chain identity records
- Minting transactions and patterns
- Transfer events between wallets
- Reputation summaries
- Agent-to-agent interaction patterns

This provides empirical evidence that blockchain-based agent identity is being actively adopted and can support decentralized trust systems for AI agents.

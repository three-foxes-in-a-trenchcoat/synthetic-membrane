# The Agentic Web — Open, Decentralized Agent Ecosystems

## The Vision

The transition from a human-centered internet to an **agentic web** — a decentralized digital ecosystem where AI agents from different users, organizations, and runtimes discover one another, negotiate task boundaries, and delegate work across open technical and social surfaces at scale.

## Evolution: Internet → IoT → Agentic Web

| Phase | What Connects | Primary Function | Key Limitation |
|-------|---------------|------------------|----------------|
| Internet | Computers | Information exchange | Human-operated |
| IoT | Devices | Perception/sensing | Passive, centralized |
| Agentic Web | AI Agents | Perception, reasoning, action | Fragmented, trust gap |

## Key Requirements

### 1. Agentic Identity and Personhood ([[agent-identity-cryptography]])

Agents need persistent identity — continuity as a social entity, not a resettable function call. Three approaches exist: PKI ([[ans]]), blockchain ([[erc-8004]]), and protocol-level ([[aip-protocol]]).

### 2. Open Collaboration ([[agent-discovery-registry]])

Agents participate in open networks, not just closed ecosystems. Discovery is prerequisite to collaboration.

### 3. Lifelong Evolution ([[evovagent]])

Agents improve across task performance, communication, and collaboration over time — not just within sessions.

### 4. Distributed Legal Infrastructure ([[distributed-legal-infrastructure]])

Five-layer framework: identity → cognitive constraints → adjudication → market regulation → portable institutions.

### 5. Interoperability ([[physical-ai-agents]])

Semantic interoperability across heterogeneous agents — the membrane provides this coordination fabric.

## Membrane as Agentic Web Infrastructure

The synthetic membrane is the coordination layer of the agentic web:
- **Layer 0** (Identity + Discovery): Who's there and what can they do?
- **Layer 1** (Permeability): What information crosses between agents?
- **Layer 2** (Shared Memory): What do agents share?
- **Layer 3** (Coordination): How do agents collectively decide?
- **Layer -1** (Governance): What rules govern the ecosystem?
